# Утилиты проекта детектора эпилепсии у крыс

## analyze_excel.py
Анализ структуры Excel файлов, включая информацию о листах, столбцах и типах данных.

## convert_excel_to_txt_corrected.py
Преобразование Excel файлов в текстовые файлы с корректным форматом для последующей обработки.

## organize_files.py
Организация файлов по папкам животных на основе идентификаторов.

## reorganize_data_structure.py
Реорганизация структуры данных в проекте, создание правильной структуры с папками сессий для каждого животного.

## tensorboard_analysis

### extract_tensorboard_data.py
Скрипт для извлечения данных из логов TensorBoard для анализа экспериментов. Извлекает метрики обучения, валидации и тестирования для создания отчетов по каждому эксперименту.

### extract_tensorboard_comprehensive.py
Скрипт для комплексного извлечения данных из логов TensorBoard. Создает сводные отчеты с таблицами сравнения различных моделей по ключевым метрикам.

### analyze_tensorboard_detailed.py
Скрипт для детального анализа структуры логов TensorBoard. Показывает все доступные метрики и их значения для понимания структуры данных.
